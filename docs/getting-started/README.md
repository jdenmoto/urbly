# Getting started

## Qué leer primero
1. `setup.md` para instalar y correr el proyecto
2. `current-status.md` para entender qué está implementado y qué sigue
3. `project-overview.md` si necesitas contexto de producto

## Documentos en esta carpeta
- `setup.md`: instalación y configuración mínima
- `current-status.md`: estado real del proyecto y deudas activas
- `project-overview.md`: resumen corto del producto
- `firebase-setup.md`: detalle extendido de Firebase
- `env.md`: variables de entorno
- `functions.md`: funciones y deploy
- `google-maps.md`: integración Maps
- `seed.md`: seed y utilidades de datos
