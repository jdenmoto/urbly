# Urbly v2

Este directorio reúne la documentación de rediseño de producto para la siguiente etapa de Urbly.

## Documentos
- Design doc estratégico: `docs/implementation/urbly-v2/design.md`
- El plan operativo vigente vive en: `docs/implementation/current-implementation-plan.md`

## Objetivo
Mover Urbly desde un panel administrativo orientado a módulos internos hacia un sistema operativo de servicio para tres actores:
- empresa
- técnico
- cliente

Y dejar la IA integrada dentro del flujo operativo, no como una feature aislada.
