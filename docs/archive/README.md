# Archivo histórico

Esta carpeta guarda documentación útil como contexto, pero no es la mejor fuente para entender el estado actual del proyecto.

## Contiene
- `plan/`: especificación y roadmap histórico por temas
- `plans/`: planes de ejecución puntuales de tramos anteriores

## Regla práctica
Si necesitas decidir qué hacer ahora, empieza por:
1. `docs/getting-started/current-status.md`
2. `docs/implementation/current-implementation-debt.md`
3. `docs/implementation/urbly-v2/`
